> 1. Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?

1.1. Создать репозиторий на github
1.2. (опционально) Если нет ssh-ключа - сгенерировать его и указать в своем профиле github открытую часть ключа
1.3. Запустить GitBash, переместиться в каталог будущего репозитория
1.4. Скопировать URL репозитория с github в буфер обмена
1.5. Выполнить в Git Bash:
git clone <URL репозитория из буфера обмена>

> 2.Как получить на локальный комьютер изменения с github.com

Запустить GitBash, переместиться в каталог своего локального репозитория и выполнить команду:
git pull

> 3. Какую информацию нам дают команды:

> git status
Показывает состояние локального репозитория: какие файлы изменены относительно последнего коммита; какие файлы еще не добавлены к отслеживанию; какие файлы изменены, закоммичены, но не отправлены в удаленный репозиторий, и т.п.

> git log
Выведет историю коммитов.

> git remote -v
Показывает адрес удаленного репозитория, с которым связан локальный репозиторий.
